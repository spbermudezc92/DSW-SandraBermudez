Propósito del sitio

En este sitio web se encuentra la definición de los términos Flex Box, CSS Grid los cuales son utilizados para el desarrollo del lenguaje HTML5, dichos términos son conceptos básicos para el desarrollo de un sitio web

FLEX BOX

Definicion

Flexbox es un módulo de diseño de CSS3 el cual permite alterar el ancho, alto y posicionamiento de elementos de la mejor manera con el espacio del que disponemos. Dicha etiqueta fue creada para mejorar la forma en la que se hace diseño responsive, evitando así el uso de float, logrando on ello que se escribiera menos código y facilitando el posicionamiento de elementos.

Estrutura

La estructura de Flexbox se compone de contenedores padre e hijos (Contenedor-Flex y Elementos-Flex respectivamente)

- El Contenedor-Flex:

Es nuestro contenedor padre, y es él quién va a contener a todos los elementos hijos a los cuáles queremos posicionar.

- El Elemento-Flex :

Es nuestro elemento hijo, el cual utilizará todo el espacio disponible para ubicarse de acuerdo a las propiedades a las cuales este sometido junto a los demas Elementos-Flex contenidos.

CSS Grid

Definicion

El CSS Grid se puede utilizar para lograr muchos diseños diferentes. También se destaca por permitir dividir una página en áreas o regiones principales entre partes de un control construido a partir de primitivas HTML.
Con CSS grid son posibles muchos más diseños y de forma más sencilla que con las tablas. Por ejemplo, los elementos secundarios de un contenedor de cuadrícula podrían posicionarse para que se solapen y se superpongan, de forma similar a los elementos posicionados en CSS.
